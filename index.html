Gemini CLI

Gemini CLIとは

Gemini CLIは、Googleによって開発された大規模言語モデル「Gemini」をコマンドラインから対話的に利用するためのツールです。

特徴

  • ファイルシステムの操作
  • コードの生成と実行
  • Web検索
  • その他多数の拡張機能

インストール方法

Gemini CLIをインストールするには、Node.jsが事前にインストールされている必要があります。以下のコマンドを実行して、Gemini CLIをインストールします。

npm install -g @google/gemini-cli

基本的な使い方

ターミナルを開き、「gemini」と入力するだけで、すぐに会話を始めることができます。

gemini [command] [options]

VSCodeでの利用

VSCodeでGemini CLIを利用する場合は、Gemini CLI Companion拡張機能の利用が便利です。この拡張機能により、GUIを通じてより直感的にGemini CLIを操作できます。

インストール方法

  1. VSCodeの拡張機能マーケットプレイスで「Gemini CLI Companion」を検索します。
  2. 「インストール」ボタンをクリックして、拡張機能をインストールします。

VSCodeとの接続方法

VSCodeのターミナルで以下のコマンドを実行することで、Gemini CLIとVSCodeを接続できます。

gemini ide enable

主な機能

  • CLIで実行したコマンドの履歴表示
  • チャット形式での対話
  • ファイル編集時のdiff表示
